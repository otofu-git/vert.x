설치 가이드

Vert.x 로 뭔가를 하려면 먼저 설치를 해야겠죠? 다음을 보면서 설치를 해 봅시다.

배포판 얻기


Vert.x 를 가장 쉽게 얻는 방법은 배포판 바이너리를 다운로드 하는 것입니다. 아니면 소스를 직접 빌드해서 사용할 수도 있습니다. 자세한 방법은 github wiki를 참조하세요.

미리 준비해야할 것들


  • 운영체제. Vert.x 는 Linux, OSX 또는 Windows 에서 잘 작동합니다.

  • JDK. Vert.x 는 JDK 1.7.0 혹은 더 최신버전을 필요로 합니다. 오라클 공식 배포버전을 이용할 수도 있고 OpenJDK 버전을 사용해도 됩니다. JDK 실행폴더를 환경변수의 PATH안에 포함시켜주세요.

  • JRuby. Ruby 와 같이 Vert.x 를 사용하려면 JRuby를 설치하고 환경변수 JRUBY_HOME 에 JRuby가 설치된 경로를 추가합니다. 그리고 json 도 설치합니다. Ruby Gem 으로 다음과 같이 실행합니다. jruby -S gem install json Ruby 를 사용하지 않으려면 이 항목은 필요하지 않습니다.

  • Jython. Python 으로 Vert.x 를 사용하려면 Jython을 설치하고 환경변수 JYTHON_HOME 에 Jython이 설치된 경로를 추가합니다. Vert.x 에서 Jython 을 이용하지 않으려면 이 항목은 필요없습니다.

vert.x 설치


위의 준비해야 할 것들을 모두 갖췄다면 vert.x 를 설치해봅시다.

  1. 다운받은 배포판을 적당한 곳에 압축 해제합니다.
  2. Vert.x 의 bin 디렉토리를 환경변수 PATH안에 넣어주세요.

버전 확인


제대로 설치가 되었는지 확인 해 볼까요? 콘솔을 열고 다음과 같이 타이핑합니다.

tim@Ethel:~/example$ vertx version
vert.x 1.1.0.final

위와 같이 나오면 성공!

테스트


간단한 웹 서버를 작성해보면서 잘 설치되었는지 다시 확인해보도록 합시다.

아래 코드를 복사해서 server.js로 저장합니다.

load('vertx.js');

vertx.createHttpServer().requestHandler(function(req) {
  req.response.end("Hello World!");
}).listen(8080, 'localhost');

파일이 위치한 곳에서 콘솔을 열어 아래를 타이핑 합니다.

vertx run server.js

브라우저를 열고 http://localhost:8080에 접속해 봅시다.

"Hello World!" 가 나타나나요? 참 잘했어요!
